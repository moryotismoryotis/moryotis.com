# 🏛️ Moryotis — Le Village des Créateurs

**Moryotis** est un environnement web 3D immersif et interactif conçu pour connecter et mettre en valeur les artistes et les créateurs de contenu. La plateforme propose une expérience de navigation dynamique cassant les codes du site vitrine traditionnel.

---

## 🚀 Fonctionnalités Clés

* **Navigation Immersive :** Exploration fluide en 3D au sein d'un village virtuel et d'espaces d'exposition.
* **Architecture Dynamique :** Intégration complète d'une base de données (Supabase) pour la gestion des interactions et des contenus en temps réel.
* **Hub Multimédia :** Espaces optimisés pour la diffusion de scripts, de vidéos et d'assets visuels.

## 🛠️ Stack Technique

* **Frontend :** HTML5, CSS3, JavaScript (Logique de navigation 3D)
* **Backend & Database :** Supabase (Temps réel, authentification et stockage des données)
* **Déploiement :** CI/CD automatisé
